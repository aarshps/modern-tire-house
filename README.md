## 📁 Project Overview

This repository contains the source code for the Modern Tire House website, a static HTML/CSS/JavaScript project deployed to https://moderntirehouse.com.

The project uses pure HTML, CSS, and JS. It is hosted on GoDaddy cPanel Shared Hosting and uses GitHub Actions for automated deployment.

## 📚 Documentation & Billing

For detailed human-readable documentation regarding the billing strategy, renewals, and project history, please refer to the `wiki` directory in this repository.

* **Wiki Location**: [wiki/Billing-Strategy.md](wiki/Billing-Strategy.md)

Future agents working on this project should refer to `.agents/AGENTS.md` for specific instructions on billing strategies, invoice generation, and retrieving GoDaddy renewal costs.

## 🛠️ Development

1. Clone the repo:
   ```sh
   git clone https://github.com/aarshps/modern-tire-house.git
   ```
2. Open `home.html` in your browser or editor.
3. Make your changes and test locally.

For deployment details, refer to the `.github/workflows` directory.
